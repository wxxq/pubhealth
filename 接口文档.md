## 1.概述

公共医疗接口整体采用http方式，返回是json格式。返回结果的中status标识整个请求的结果，成功返回1，失败返回0，json如下：
```json
{
	"status": 1,
	"result": 200
}
{
	"status": 0,
	"result": [{
		"errorCode": 2009,
		"message": "用户已激活"
	}]
}
{
	"status": 0,
	"result": [{
		"errorCode": 1014,
		"message": "邮箱无效"
	}, {
		"errorCode": 1010,
		"message": "密码无效"
	}]
}
```
result中是成功或失败的返回值，错误返回值返回错误代码和错误信息，可以返回多个，以列表方式，成功的返回值也可以返回一个或多个。下面例子只提供一种json格式。
